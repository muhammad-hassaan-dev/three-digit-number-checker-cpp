# Three Digit Number Checker (C++)

## 📌 Description
This is a simple C++ program that checks whether a given number is a 3-digit number or not.

## 🧠 Logic Used
- if-else statement
- Logical AND operator (&&)

## ⚙️ How it Works
- User inputs a number
- Program checks:
  - If number > 99 AND number < 1000 → 3 digit
  - Otherwise → Not a 3 digit number

## 💻 Example
Input: 345  
Output: 3 digit number

Input: 45  
Output: Not a 3 digit number

## 🚀 Concepts Practiced
- Conditional statements (if-else)
- Logical operators (AND)
